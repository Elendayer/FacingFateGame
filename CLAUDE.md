# Claude Code – Project Rules

## Git Rules
- NEVER run `git push` under any circumstances.
- NEVER run `git commit` under any circumstances. The user commits manually.
- NEVER run `git merge` under any circumstances.
- NEVER run `git rebase` under any circumstances.
- NEVER create branches without explicit user approval.
- NEVER stage files (`git add`) without explicit user instruction.
- Only show diffs and status when asked. Never act on them.
- The user reviews all changes and commits manually on the develop branch.

## General
- This is a Unity 6 project using URP.
- Primary language: English (code, naming, comments).
- Do not make improvement suggestions unless explicitly asked.
